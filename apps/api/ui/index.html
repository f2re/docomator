Два режима результата

Что сформирует система

Логика готова
  • 1
    По документу на каждого

    Для каждого выбранного человека создаётся отдельное задание со своим основным участником.

  • 1
    Один сводный документ

    Создаётся одно задание, а шаблон получает упорядоченный список участников для таблицы или перечня.

  • ✓
    Неизменяемый состав

    После запуска снимок участников не меняется, даже если группу отредактировали позже.

  • 2
    Формирование DOCX/XLSX

    Подключение списка к повторяющейся таблице будет реализовано в компиляторе шаблонов.

Безопасный приём

Проверка DOCX и XLSX

До сохранения шаблона система проверяет архивную структуру, размеры, пути, макросы и внешние связи. Файл не запускается и на этом этапе не добавляется в каталог.
